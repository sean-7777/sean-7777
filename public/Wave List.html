Wave List

Usage

<wave-list width=[number] manuallinebreak?>
The wrapper for the wave list which is the main component. The attribute width should be a number which will change the size of the entire list. manuallinebreak is a boolean attribute and will override width. It will automatically set the wave listâ€™s width so that all content can fit on one line. It should be used with <br>.
<list-title width=[number] color=[string]>
Defines the title for the wave list. Argument usage exactly the same as <list-item>.
<list-item width=[number] color=[string]>
Defines a item for the wave list. Should be only used in the body of <wave-list>. width is a number that will define the width of the item. color is any valid CSS3 color keyword, hex code, or RGB/HSL function.

Examples

Basic

<wave-list>
  <list-title color="red">Sample Title</list-title>
  <list-item color="green">List Item 1</list-title>
  <list-item color="orange">List Item 2</list-title>
  <list-item color="pink">The Last List Item</list-title>
</wave-list>

Widths

<wave-list width="200"> <!-- Double width -->
  <!-- Slightly smaller width -->
  <list-title color="red" width="75">Sample Title</list-title>
  <list-item color="green">List Item 1</list-title>
  <!-- Longer item width -->
  <list-item color="orange" width="130">List Item 2</list-title>
  <list-item color="pink">The Last List Item</list-title>
</wave-list>
